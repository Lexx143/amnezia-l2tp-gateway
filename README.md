# Amnezia L2TP Gateway

Этот проект позволяет подключать устройства, не поддерживающие современные VPN-протоколы (например, старые роутеры), к серверам AmneziaVPN (протокол AmneziaWG) через классический L2TP/IPsec.

Доступно два варианта развертывания:
1. **Cloud VPS (Рекомендуется):** Использование недорогого промежуточного сервера (например, в РФ) для маршрутизации трафика роутера в зарубежный туннель.
2. **Local Docker Gateway:** Развертывание шлюза в локальной сети на Windows/Linux ПК через Docker.

---

## Вариант 1: Cloud VPS Gateway (Рекомендуется)

Этот метод не требует держать домашний компьютер включенным. Роутер подключается к промежуточному VPS-серверу, который затем направляет трафик в AmneziaWG.

### Установка:
1. Арендуйте любой дешевый VPS с ОС Ubuntu (например, в Яндекс.Облаке).
2. Загрузите на сервер скрипт установки:
   ```bash
   wget https://raw.githubusercontent.com/Lexx143/amnezia-l2tp-gateway/main/cloud_vps/install.sh
   chmod +x install.sh
   sudo ./install.sh
   ```
3. Скопируйте ваш профиль клиента `AmneziaWG` в файл `/etc/amnezia/amneziawg/awg0.conf`.
4. Обязательно добавьте следующие строки в секцию `[Interface]` вашего `awg0.conf`, чтобы настроить маршрутизацию:
   ```ini
   PostUp = ip rule add from 10.0.1.0/24 table 123; ip route add default dev awg0 table 123
   ```
5. Включите туннель:
   ```bash
   sudo systemctl enable --now awg-quick@awg0
   ```

### Подключение клиентов:
* **Без ключа IPsec (для роутеров):** Тип L2TP, IP адрес вашего VPS, логин `router`, пароль `secret123`.
* **С ключом IPsec (для телефонов/ПК):** Тип L2TP/IPsec PSK, общий ключ `amnezia123`, логин `Evgenii`, пароль `secret123`.

## Вариант 2: Local Docker Gateway

Если вы хотите использовать свой домашний ПК (Windows/Linux) в качестве шлюза в локальной сети.

### Установка:
1. Скопируйте настройки из приложения AmneziaVPN (вкладка "Поделиться" -> AmneziaWG) и вставьте их в файл `awg0.conf`. Обязательно добавьте маршрутизацию: `PostUp = ip rule add from 10.0.1.0/24 table 123; ip route add default dev awg0 table 123`.
2. В файле `chap-secrets` задайте логины и пароли в формате: `пользователь * пароль *`.
3. Запустите скрипт `start.bat` (для Windows) или выполните `docker-compose up -d --build` (для Linux).

### Подключение роутера:
Создайте L2TP клиент на вашем роутере и укажите локальный IP-адрес ПК, на котором запущен Docker.

> **Внимание:** В этом режиме компьютер должен работать круглосуточно.
