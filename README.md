# AmneziaWG to L2TP Docker Gateway

Уникальное решение для обхода ТСПУ Роскомнадзора для старых или "глупых" роутеров (например, 4G-роутеров на даче), которые не поддерживают современные протоколы маскировки, а их родные VPN (OpenVPN, L2TP/IPsec) жестко блокируются мобильными операторами по сигнатурам.

**Архитектура:**
`Уличный 4G-роутер` ---> `(Чистый L2TP UDP 1701)` ---> `Домашний ПК (Docker шлюз)` ---> `(AmneziaWG)` ---> `Машина VPS`

## Шаг 0: Настройка сервера (VPS)
Вам не нужно писать сложный код для сервера! 
1. Арендуйте любой зарубежный VPS (Ubuntu/Debian).
2. Скачайте официальное приложение **AmneziaVPN** на свой компьютер.
3. Добавьте ваш сервер в приложение (потребуется IP и SSH-пароль/ключ).
4. Выберите протокол **AmneziaWG** и нажмите "Установить".
5. После установки перейдите в настройки сервера -> "Поделиться подключением" -> "Для клиента WireGuard / OpenVPN / Настроить вручную".
6. Скопируйте полученный текст — это ваш клиентский конфиг.

## Шаг 1: Настройка шлюза (Домашний ПК Windows/Mac/Linux)
1. Склонируйте этот репозиторий.
2. Скопируйте файл `awg0.conf.example` в `awg0.conf`.
3. Вставьте в `awg0.conf` текст клиентского конфига, который вы скопировали в приложении AmneziaVPN.
4. Запустите шлюз:
   - На Windows: Убедитесь, что запущен Docker Desktop, и просто дважды кликните `start.bat`.
   - На Mac/Linux: В терминале выполните `docker-compose up -d`.

## Шаг 2: Проброс портов (Port Forwarding)
Зайдите в админку вашего домашнего роутера (где стоит ПК со шлюзом).
Создайте правило проброса портов:
- **Внешний порт:** `1701` (UDP)
- **Внутренний порт:** `1701` (UDP)
- **IP-адрес:** Локальный IP вашего ПК (например, 192.168.1.50).

## Шаг 3: Настройка клиентского 4G-роутера
Зайдите в админку роутера, который вы хотите подключить к VPN, и создайте L2TP-клиента:
- **Адрес сервера:** Ваш домашний внешний ("белый") IP-адрес.
- **Логин:** `router` (настраивается в файле `chap-secrets`)
- **Пароль:** `secret123` (настраивается в файле `chap-secrets`)
- **Шифрование (IPsec):** ОТКЛЮЧИТЬ. Нам нужен "чистый" L2TP. Мобильные операторы РФ активно блокируют IPsec-рукопожатия (порты 500/4500), но часто пропускают чистый L2TP (1701).

Готово! Весь трафик 4G-роутера теперь завернут в непробиваемый туннель AmneziaWG.
